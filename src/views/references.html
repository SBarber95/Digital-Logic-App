Boolean Theorem Reference

Operations with 0 and 1

X + 0 = X

X + 1 = 1

X * 1 = X

X * 0 = 0

Idempotent Laws

X + X = X

X · X = X

Involution Law

(X’)’ = X

Commutative Laws

X + Y = Y + X

XY = YX

Associative Laws

(X + Y) + Z = X + (Y + Z) = X + Y + Z

(XY)Z = X(YZ) = XYZ

Laws of Complimentarity

X + X’ = 1

X · X’ = 0

Distributive Laws

X(Y + Z) = XY + XZ

X + YZ = (X + Y)(X + Z)

Simplification Theorems

XY + XY’ = X

X + XY = X

(X + Y’)Y = XY

(X + Y)(X + Y’) = X

X(X + Y) = X

XY’ + Y = X + Y

DeMorgan's Laws

(X + Y + Z +...)’ = X’Y’Z’...

(XYZ...)’ = X’ + Y’ + Z’...

Duality

(X + Y + Z +...) = XYZ...

(XYZ) = X + Y + Z +...

Theorem for Multiplying Out and Factoring

(X + Y)(X' + Z) = XZ + X'Y

XY + X'Z = (X + Z)(X' + Y)

Consensus Theorem

XY + YZ + X'Z = XY + X'Z

(X + Y)(Y + Z)(X' + Z) = (X + Y)(X' + Z)
